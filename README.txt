Educulas - School Campus Management system

This project is under development,built using Zend framework and Doctrine 2
Working currently on the attendance module, trying to integrate jqgrid with zend and create an attendance sheet.
Created custom jquery view helpers ( Multiple list autocomplete and a calendar as navigation link)
Twitter Bootstrap CSS framework has been linked to Zend, layout is created.

All the controllers are in application/controllers directory,
The Entity classes are in Campus library
The CRUD code can be found in service folder in Campus/Entity/Service
